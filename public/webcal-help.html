Manual Calendar Subscription Instructions

  1. Download the calendar file using the button above
  2. Open your calendar application (Google Calendar, Apple Calendar, Outlook, etc.)
  3. Look for an option to "Add Calendar", "Subscribe to Calendar", or "Import"
  4. Select the downloaded .ics file
  5. Follow your calendar application's prompts to complete the subscription

Note: For a true subscription (automatic updates), your calendar app must support the "webcal://" protocol or subscription to ICS files from a URL.
